# Future plan: Demo prep — game selection for the PC server

**Why it matters:** The end-of-month demo streams a real playtest build from a PC server. The current PC server hardware is modest, so the demo game must run well on it and ideally show off something more than a trivial scene.

## Current understanding (Sync 3 — Nate, David K)
- PC Server "Take Home" just launched — you can play on a PC server today.
- Current spec: a **T4 (Tesla 4) GPU, 8 vCPU**, roughly equivalent to **Xbox Series S**. Planned upgrade to a **B4** next year.
- So the demo title **must not be a graphics powerhouse**. David K: "let's do better than" a spinning triangle/cube — he has access to **Dolphin** and may get a package running; the proposal mentions AI, so something a bit more impressive than a rotating cube.

## Options
- Pick a lightweight but visually credible title that runs comfortably on a T4 / Series S–class server (e.g. Dolphin-based package).
- Validate end-to-end (ingest → offering → stream) with that title before the demo.
